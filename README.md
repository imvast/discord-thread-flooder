### Setup
+ `pip install` the needed modules
+ `python3 mass.py`
+ + Once you get it running do `[prefix]threadspam` in the channel you want to mass create threads in.

---------------------------------------

### Features
* User friendly interface
* Proxy support (http/socks4/socks5)
